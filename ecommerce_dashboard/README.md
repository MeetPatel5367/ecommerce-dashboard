# Enterprise E-commerce Research Analytics Platform

## Project Overview
This project is a professional and interactive e-commerce analytics dashboard developed using Streamlit. It is designed to analyse processed sales and customer data and present business insights in a clear, modern, and research-oriented format.

The dashboard helps users understand revenue trends, customer spending behaviour, segment distribution, and data quality through interactive charts, KPI cards, and downloadable outputs.

---

## Objectives
The main objectives of this project are:

- To analyse monthly e-commerce revenue
- To identify high-value customers
- To present customer segmentation insights
- To create an interactive business intelligence dashboard
- To demonstrate a practical data analytics and reporting workflow

---

## Features
This dashboard includes the following features:

- Year-wise sales analysis
- KPI cards for total revenue, average revenue, best month, and total customers
- Monthly revenue trend analysis
- Quarterly revenue summary
- Month-on-month growth analysis
- Top customers by spending
- Customer detail table
- Customer segmentation analysis
- Data quality summary
- CSV download option for processed outputs
- Technical methodology section
- Future enhancement scope

---

## Technologies Used
The project is built using the following technologies:

- Python
- Streamlit
- Pandas
- NumPy

---

## Dataset
The dashboard uses the following processed datasets:

- `gold_monthly_sales.csv`
- `gold_customer_summary.csv`

These files are stored inside the `data` folder.
